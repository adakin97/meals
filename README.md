# Meals

A shared weekly meal planner for Alex and Lauren.

**Live app:** https://adakin97.github.io/meals

## Features

- Weekly planner — assign a meal to each day and who's cooking
- Meals library — save meals with their ingredients
- Real-time sync — changes appear instantly on both devices
- Mobile-friendly — add to home screen for app-like experience

## Stack

- Vanilla HTML/CSS/JS (single file)
- Firebase Firestore (real-time database)
- GitHub Pages (hosting)

## Usage

Open the link on any device. No login required — just share the URL.
